1. Activate server using terminal
    - Go to Bin and search for PostgreSQL
        $cd c:\Program Files\PostgreSQL\18\bin 
    - Execute psql using username & pass 
        default is postgres
        $psql -U postgres
        $ Password:
    - Working on Database:
        list db:      $\l
        use db:       $\c databaseName
        list tables:  $\d
        select data:  $ select * from tableName

2. Create Database with GUI:
    - Databases > Create > Database OR ALT+SHIFT+N
